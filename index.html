Intégrer votre dashboard Power BI

Cette page vous permet d'intégrer votre dashboard Power BI publié en ligne directement dans ce fichier HTML — exactement comme le fait ATNI (Access to Nutrition initiative) sur son site web.

1
Publiez votre rapport sur Power BI Service
Dans Power BI Desktop : Fichier → Publier → Publier sur Power BI (nécessite un compte professionnel ou Microsoft 365)
2
Obtenez le lien d'intégration
Dans Power BI Service : ouvrez votre rapport → Fichier → Incorporer le rapport → Site web ou portail → copiez le lien src de l'iframe
3
Collez le lien ci-dessous
Le dashboard Power BI apparaît ici avec toutes ses interactions (filtres, slicers, drill-down)
Aperçu du dashboard intégré
Votre dashboard Power BI apparaît ici avec toutes les interactions
📊

Collez votre lien Power BI ci-dessus et cliquez "Afficher le dashboard" pour voir votre rapport interactif ici.

Alternative : Google Looker Studio (sans compte pro)

Si vous n'avez pas de compte Power BI professionnel, Google Looker Studio offre la même fonctionnalité gratuitement avec un compte Gmail :

1
Créez votre dashboard sur lookerstudio.google.com en connectant votre fichier Google Sheets
2
Partagez → Intégrer le rapport → copiez le lien iframe et collez-le ci-dessus
